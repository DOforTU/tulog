## 프로젝트 개요

### 프로젝트명

**TULOG** - 개인 및 팀 블로그 플랫폼

### 프로젝트 배경

공부한 내용을 직접 운영하는 블로그에 게시하고 싶어서 시작한 프로젝트입니다. 개인뿐만 아니라 팀 단위로도 지식과 경험을 공유할 수 있는 플랫폼을 목표로 합니다.

### 기술 스택

-   **Backend**: NestJS, TypeScript, PostgreSQL, TypeORM
-   **Frontend**: Next.js 15, React 19, TypeScript, Tailwind CSS
-   **Authentication**: JWT, Google OAuth 2.0
-   **Development**: ESLint, Prettier, Jest

### 프로젝트 구조

```
TULOG/
├── tulog/              # 프로젝트 메인 (기획, 문서, 문제 기록)
├── tulog-server/       # Backend API (NestJS)
└── tulog-ui/          # Frontend (Next.js + Vue.js)
    ├── ui-next/       # Next.js 애플리케이션
    └── ui-vue/        # Vue.js 애플리케이션
```

## 핵심 기능

### 1. 사용자 관리 시스템

#### 1.1 인증 및 권한

-   **OAuth 로그인**: Google OAuth 2.0을 통한 소셜 로그인
-   **JWT 토큰**: Access Token과 Refresh Token 발급
-   **권한 관리**: USER, ADMIN, SUPER_ADMIN 역할 구분
-   **소프트 삭제**: 계정 탈퇴 시 데이터 보존

#### 1.2 사용자 프로필

-   **기본 정보**: 이메일, 닉네임, 프로필 사진
-   **정보 수정**: 닉네임, 프로필 사진 변경
-   **비밀번호 관리**: 로컬 로그인 시 비밀번호 변경/찾기

#### 1.3 사용자 상호작용

-   **팔로우 시스템**: 다른 사용자 및 팀 팔로우
-   **차단 기능**: 원치 않는 사용자 차단
-   **신고 시스템**: 부적절한 사용자 신고 (개발자 메일로 접수)

### 2. 블로그 포스팅 시스템

#### 2.1 글 작성 및 관리

-   **마크다운 지원**: 마크다운 언어로 글 작성
-   **이미지 업로드**: 다중 이미지 업로드 (첫 번째 이미지가 대표 이미지)
-   **임시 저장**: 작성 중인 글 임시 저장
-   **공개 설정**: 공개/비공개 상태 설정
-   **태그 시스템**: 최대 10개 태그 지원

#### 2.2 글 조회 및 상호작용

-   **공개 글 조회**: 모든 사용자가 공개 글 조회 가능
-   **즐겨찾기**: 관심 있는 글 즐겨찾기
-   **좋아요**: 글에 좋아요 표시
-   **차단**: 특정 블로그 차단

### 3. 팀 시스템

#### 3.1 팀 생성 및 관리

-   **팀 생성**: 최대 3개 팀 참여 가능
-   **팀원 관리**: 최대 10명까지 팀원 구성
-   **팀장 권한**: 팀원 초대, 강퇴, 권한 위임
-   **팀 상태 설정**: 선택 수락 / 초대만 가능 모드

#### 3.2 팀 상호작용

-   **팀 조회**: 다른 팀 검색 및 조회
-   **팀 참여**: 팀 가입 신청 및 수락
-   **즐겨찾기**: 관심 있는 팀 즐겨찾기
-   **차단**: 원치 않는 팀 숨김 설정
-   **신고**: 부적절한 팀 신고

#### 3.3 팀 권한 관리

-   **팀장 자동 이전**: 팀장 탈퇴 시 최근 접속자에게 권한 이전
-   **팀 자동 삭제**: 팀원이 1명일 때 탈퇴하면 팀 자동 삭제

### 4. 댓글 시스템

#### 4.1 댓글 작성

-   **공개 글 댓글**: 모든 사용자가 공개 글에 댓글 작성 가능
-   **비공개 글 댓글**: 본인 글의 경우 비공개 글에도 댓글 작성 가능

#### 4.2 댓글 관리

-   **본인 댓글**: 수정 및 삭제 가능
-   **타인 댓글**: 본인 글의 경우 타인 댓글 삭제 가능

### 5. 좋아요 및 즐겨찾기

#### 5.1 좋아요 시스템

-   **좋아요 표시**: 공개 글에만 좋아요 가능
-   **좋아요 필터링**: 좋아요한 글만 조회 가능

#### 5.2 즐겨찾기 시스템

-   **블로그 즐겨찾기**: 관심 있는 블로그 즐겨찾기
-   **팀 즐겨찾기**: 관심 있는 팀 즐겨찾기

### 6. 알림 시스템

-   **팔로우 알림**: 팔로우한 사용자/팀의 공개글 알림
-   **마케팅 알림**: 마케팅 동의 사용자에게 광고 알림

### 7. 검색 시스템

-   **통합 검색**: 블로그 글, 사용자, 팀 통합 검색
-   **유사도 검색**: Transformer 기반 유사도 검색
-   **태그 검색**: 태그 기반 글 검색
