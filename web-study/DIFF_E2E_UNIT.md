**E2E vs 단위 테스트 차이점**

| 구분          | 단위 테스트      | E2E 테스트        |
| ------------- | ---------------- | ----------------- |
| **범위**      | 개별 함수/클래스 | 전체 애플리케이션 |
| **실행 속도** | 빠름             | 느림              |
| **격리성**    | 높음 (모킹 사용) | 낮음 (실제 환경)  |
| **신뢰도**    | 낮음             | 높음              |
| **디버깅**    | 쉬움             | 어려움            |
| **목적**      | 로직 정확성      | 전체 시스템 동작  |
